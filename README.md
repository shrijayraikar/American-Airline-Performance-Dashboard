# American-Airline-Performance-Dashboard
**Power BI dashboard analyzing domestic and international flight performance for American Airlines (Jan–Feb 2025).**

Data Cleaning:

- Removed duplicate and null flight records
- Standardized route and airport naming (city pairs, region tagging as Domestic/International)
- Corrected date/time formatting for delay calculations
- Created calculated columns for profit, profit %, and delay categorization

Key Metrics:

- $12.46M total revenue across 187 flights, 10 routes
-  36K total passengers
-  90 min average delay D
-  FW-SEA identified as most profitable route ($273K profit)

Features:

- Geographical route mapping
- Day-to-day revenue & profit trend analysis
- Route-wise revenue/profit comparison
- Delay analysis by route
- Route profitability breakdown (profit %, avg delay, flight count)
- Interactive filters by Region and Route
